# Phase Roadmap

## Phase 1 - Data Platform Foundation

Build the core platform:

- Source systems.
- Kafka.
- Debezium.
- Airflow.
- Spark.
- Iceberg.
- Object storage.
- Hive Metastore.
- SQL access tools.

## Phase 2 - Governed Semantic Layer And Hybrid Readiness

Build governed structured access and prepare hybrid AI contracts:

- Denodo data sources.
- Denodo business views.
- Spark Thrift access.
- Approved metric templates.
- MCP tool contracts.
- Hybrid router rules.
- Document RAG source inventory.
- Answer validation contract.

## Phase 3 - AI / RAG Runtime

Implement AI access:

- Prompt UI.
- LLM reasoning.
- MCP server tools.
- Governed Denodo query execution.
- Document embedding/indexing.
- Hybrid routing.
- Grounded answer formatter.
- Answer validator.

## Phase 4 - BI And Enterprise Consumption

Enable governed consumption:

- Power BI semantic consumption.
- Prompt-assisted BI requirements.
- Dashboards.
- Audit reports.
- Business user enablement.

## Phase 5 - Hardening And Operations

Prepare for enterprise operation:

- Monitoring.
- Alerting.
- Security hardening.
- Compliance evidence.
- Access review.
- Performance tuning.
- Operational runbooks.
